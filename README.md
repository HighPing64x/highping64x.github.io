古神纪元■■■年，卷王-始祖形态在卷星诞生。

古神纪元114514年，古神纪元结束，旧日纪年开始。

旧日纪年514年，卷王-创新形态诞生。

旧日纪年520年，卷王-创新形态与始祖形态意见不合，分为两国并立即攻打始祖形态所在的古都。始祖形态被一块重石击中后脑，因”妈妈生的”而昏迷，并消失在人们的视野之中。

旧日纪年666年，卷王-创新形态进化为卷王之一形态。

旧日纪年1145年，卷王之一形态在看猎奇视频时被武器A精神污染，精神失常，变成卷王-武器A形态(又称武器卷)。<br>武器卷疯狂攻击邻国，旁国“纯鹿人国”节节败退，其国王-卷王路人甲在外求无援的情况下自缢而亡。
                
旧日纪年1919年，卷王之二形态诞生。

旧日纪年1920年，卷王之二形态所带领的簧斤军击溃武器卷大军，武器卷失去民心，在愤怒中选择与簧斤军同归于尽，在自爆中消失，卷王之二形态身受重伤。

旧日纪年1924年，卷王之二形态在重伤痊愈后意外打通了任督二脉，进化为卷王之三形态。

旧日纪年1949年，卷王之三形态独自飞入太空开疆拓土，且因为太空中的 嘎玛射线暴 进入深度昏迷，漂流在太空之中。、

旧日纪年5000年，旧日纪年结束，黑暗年代开始。


黑暗年代元年，在太空中的嘎玛射线多年刺激与吸收下，卷王之三形态变异为变异卷王，进入卷星大气层，在114km高空中将自身进行二维展开，包裹整个卷星。暗能量覆盖入卷星，侵蚀进卷星岩石层，使亡灵复苏，变为黑暗亡灵，卷人无处可逃。<br>这是老卷编年史上的第一个大事件，称作[卷星长夜]。
              
黑暗年代100000年，卷星上没有了任何生物，所有生灵全部灭绝，包括自然界内的一切动植物，真菌，以及细菌病毒甚至芽孢，文明的种子彻底断绝。

黑暗年代114514年，上方二维展开的变异卷王突然异动，遁逃入太空中，文明的种子终于有了喘息的机会。变异卷王在太空中抖动，吸收了几十万年的嘎玛射线，早已使变异卷王无比强大，只见变异卷王如同细胞分裂一般，裂为了两半。一半回到了卷星，但又不知踪迹，另一半飞向无尽的星空。<br>经此一事，埋藏了很久的文明之种也重新显现。
              
黑暗年代114514年，黑暗年代结束，恶臭纪元开始。


恶臭纪元元年，卷王之恶臭形态诞生，建立homo卷国。

恶臭客元100年，卷王之恶臭形态因感情关系自缢，homo卷国灭亡，进入45年的无国时期，人口锐减，无声的威胁也在来临。

恶臭寄元1145年，恶臭大陆沉没，在水下形成大量可燃冰，后人称此地为：百慕大三角，致使homo子民消失殆尽，恶臭寄元结束，无尽星空来临。

无尽星空1年，卷王之寰宇形态来临。没有人见过他，因为都没能逃出魔爪了。他来临时，天昏地暗，乌云密布，却无人知晓。

无尽星空10000年，无人幸存，万物殒灭，强大的嘎玛射线使其近乎疯狂，嘎玛射线使卷星的所有生物都进入内卷状态，无法进行反击。

无尽星空1027年，一位伟大的人计算出寰宇卷王会像黑洞一样蒸发，终有一天会消失，但那个人很快就死了，连名字都没留下，但是这个预言却支撑着人们。

无尽星空■■■年，寰宇卷王的嘎玛射线使长眠在地下的卷王-始祖形态苏醒，并靠嘎玛射线将自己强化为了卷王-究级始祖。

寰宇卷王与究级始祖的大战，终于开始了，不过万招，寰宇卷王便被一张长二向简满分试卷击中，灰飞烟灭，而究级始祖也进入了休眠，守护卷星安宁。

无尽星空年是之前所有纪元中最长的，至少大于1027年，具体未知。进入徐坤公元。


徐坤公元2年半，卷王一爱坤形态、卷王—徐坤形态、卷王—神坤、卷王—黑子相继出现      (因战乱，画像皆丢失，无从考证)，建立了爱坤鸡朝。

徐坤公元1451年，卷王—yee形态灭亡了爱坤鸡朝，销毁了许多画像、典籍，很多珍资料也自此消失，史称“你妈不填”(女娲补天?)

徐坤公元1751年，长达300年的混战结束，和平的田园时代来临。徐坤公元结束，进入田园纪年。


田园1年，科技发展迅速，人民幸福祥和，

田园50年，科技发展至能使卷王升维、降维能力。

田园100年，科技发展至可自由制造卷子纠缠能力，武力逐渐被遗忘。

田园178年，寰宇神教召唤寰宇卷王，信仰寰宇卷王的信徒突起，准备献祭1024座大城市重新召唤寰宇卷王，由于军队战斗力下降以及寰宇卷教的信徒武器强大，军队无力反抗，节节败退。

田园178年6月6日06:06:06，寰宇卷王再次出现，卷星进入第二个黑暗年代，史称”六六神变”。   寰宇卷王自称获得了克苏鲁主神的帮助，获得了无穷的力量，如今连始祖态的究级张卷也对他无可奈何。

田园时代结束，无尽星空开始，世界再度陷入无限的混乱之中。


无尽星空元年，究级始祖卷感应到卷星的危机，再度苏醒，两大卷神对面，天雷滚滚，大地颤动，众星归位，一瞬间卷人们都消失于无形之中，不知是隔离在外还是被冲击波震得灰飞烟灭。

无尽星空2年，这场大战持续了两年半，双方不分上下，大地震裂，岩浆涌出，世界大乱。

无尽宸宇12年半，大战惊扰了一个神秘的人，那人在虚空之中游荡，感应到卷星上可毁灭星球的巨大冲击波，一惊，飞速冲向卷星，此时卷星已成了一片火海。

无尽寰宇13年，那个超脱自然的人来到卷星上空。那个人仅仅两招，便将两个卷神全部败，又一招，抛出一个无法用词汇来描述的至高圣物，就连卷星最渊博的人也无法想象。仅是击中地面，大地竞并未震动，只见地面上的神秘物体融入大地，霎时间，万物围绕击中点复苏起来，大地恢复生机。那个人见此，带走了寰宇卷王，只听重伤倒地寰宇卷王瞪着那个人，不可置信地说:”你....就是...蔡..徐...”但没等寰宇卷王说完，那人便带走了寰宇卷王，从此一去不反。

无尽寰宇结束，公元年开始。


公元二年，万物回春，人们在那个战场中心建立了一座神殿，但技术一直倒退回到初始时代，先前的大多文献，古籍，科技都消失了，一切都要从头再来。

公元一百年，在那座神殿里，一朵如同那个神秘物体的花朵慢慢开放...


公元300年，两百年过去了，之前发生的天地危机早已无人知晓，科技由于已经无人传递下去而跌落回旧石器时代，以前的辉煌只能重新开始书写，卷星的星际闻名的建筑—上帝之书也已经残破不堪，辨认不出了。

公元350年，出现了万年难遇的大水灾，人口锐减至三千万，达到了无尽寰宇时的水平，但有不少强大科技被冲刷出来，为后来的技术爆炸埋下了伏笔。

公元1145年，世界文明已经达到了铁器时代，出现三大文明：卷王国、坤王国与奄奄一息的“我的审国”。卷王国主攻于科技，在科技上极其强大；坤王国热衷于艺术，世称优雅之国。而“我的审国”，由陈紫箔领导，当权朝政昏庸腐败，武将文将贪财无能，濒临灭亡。

公元1170年，我的审国首都被愤怒的人民踏平，分裂为十七国。

公元1171年，卷王国在科技上取得了重大突破，使星舰可达到15%光速，武器强度有史诗级突破，进入科技元年。


科技元年，卷王国对坤王国进行军事攻击，并计划在攻占整个卷星后向周围的星域扩张领地。

科技3年，卷王国所有星舰在一瞬同全部消失，卷王国首领章加炫认为是坤国盗窃，但坤国首领却极力否认。

科技纪元12年，坤国灭亡，坤国现任首领刘七微归降，卷国张嘉轩御驾亲征，前往星空。

科技纪元15年，刘七微卧薪尝胆，攻占卷都，重新建立坤国，并驱遂张嘉轩等人的星舰，使其流落太空，张嘉轩等人的星舰以最大速度远离卷星，从此杳无音信。


科技纪元376年，在坤国的精心治理下，经济飞速发展，科技技术极速发展，已经制作出50%光速的星舰，进入了星空时代，领土已经达到了卷星方圆3光年。

科技纪元1145年，有人计算出每经过1200年，都会有一次天灾，轻则天地震颤，重则文明重启，种类多变，时而水灾，扬是国家争斗，在发布完这条消息之后，这位伟大的坤学家服毒自尽了，会社会震动。但仍有不少人不相信。这个预言在随后的历史中如同诅咒一般，曾多次置卷星于水火之中。

科技纪元2345年，1200年到了，卷星笼上了一层阴霾，而到来的不是天灾:而是一条500万光年远的地方的一条宇宙级广播：

由于广播大远，只能传递一点，其余的在经过星际尘埃方时就已不知所踪了。社会陷入了恐慌，官方只好出武力镇压。

科技纪元9223372036854775807年，也就是264-1-1年，远古时代遗留下来的信息们在提醒人们要更换纪元了，但岁月使人们都认为这只是个传说，无人在意，甚至有些人还在好奇与期待。

科技纪元9223372036854785808年新年的第一天，一层阴云出现在首都上空，只，是整个星球上空。在所有人惶恐的目光中，

现任领袖与星舰————应该是星球，开启了谈话。

 *清҉理҉人҉：宇宙的仆人，也是宇宙的清理人。*

旋即，星舰的周围出现了一个小光点，停留在上空，极其微小，也求其恐怖。突然星舰消失了，无影无踪。这意味着什么？超光速。卷人族用尽千百万年也没有钻透的障壁，被他们打破了。

光点，以自由落体的方式落向卷星。

领袖惊慌地命令开启星障，这是星球的全方面装甲，坚硬无比，一旦开启就是宇宙级的灾难也无法伤到卷都的分毫。但光点却轻而易举的穿透了。光点突然在几秒内加速到了光速，向地面冲去。通过质能方程，光点穿过整个星球之后，竟只是一个普通的铁球，然后马上消失在原地，不久卷星竟发出了耀眼的白光，产生大量的冲击波向四周散去，旋即消失在了原地，一粒灰尘都没有剩下。

结局-清理(坏结局)






重生纪元元年，卷星上的生灵纷纷苏醒。

生机盎然，一片祥和。卷星千万年前的篮球之种已经萌发，抵消了神之铁球的一次攻击并将整个卷星带入虚数空间之中，卷星上所有生灵都重生了一次。
